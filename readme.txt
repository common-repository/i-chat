Allows you to add live chat functionality. Chat with your web visitors if they have any queries or question can be resolved on the fly.

== Description ==

I Chat allows you to add Live Chat functionality to your website. Once install it automatically adds the Chat Window on every page of your website. No need to edit your themes or anything. I Chat doesnt use any third party connections all the data and conversation messages are stored on your database. You can directly communicate with your vistors and resolve there queries, This will increase your interaction with web visitors.

Free Version Features

* I Chat adds a small chat window, which wont take a large space on the web page
* I Chat takes users details like Name, Email before starting the chatting session, and stores it in your database
* Chat window can be minimised or closed
* Live Chat with your web visitors
* Chat with multiple visitors simultaneously
* Chat messages will render without reloading the window
* Admin can reply to the visitors from admin pannel
* I Chat shows the list of all the chat sessions click any one of them Chat Window will be visible
* Admin can directly reply from the same window
* If new message came from any other visitor admin is able to see on the list
* Admin can directly click on that chat session and chat window for that vistor will be open
* No need to go out of the window, You can reply to the messages in the same window


I Chat Pro

* Set colors of chat window as per your theme
* Set chat window Icon
* Set the chat window Title
* Settings to open the chat window after some time period
* Add a default message to show up after certain period of time
* Visitors will be notified by Desktop notification once the received a message.
* Admin Gets an android application so that you dont need to login to your website to chat with your visitors, you can talk to them from wherever you want
* Push Notifications for Admin chat android application so you wont miss any visitor



== Installation ==

1. Download the latest version from here, unzip it, and upload the extracted files to your plugins directory.
2. Activate the plugin via your Wordpress admin panel.
3. Head over to Settings -> I Chat.
4. You will get the list of all the visitors just click the row and start chatting with them.
